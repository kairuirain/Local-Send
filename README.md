# LocalShare - 局域网文件分享中心

一个轻量级、易用的局域网文件传输解决方案，支持多设备间的文件快速分享与下载。

## 📖 项目简介

LocalShare 是一个基于 Flask 开发的局域网文件分享工具，专为家庭和办公环境设计。它提供简洁的 Web 界面，支持文件拖拽上传、批量下载、中文文件名等功能，无需复杂的配置即可在局域网内快速搭建文件共享服务。

### 主要特点

- 🚀 **即开即用** - 无需配置，双击启动即可使用
- 🎨 **现代界面** - 响应式设计，支持多端访问
- 📁 **文件管理** - 支持上传、下载、删除等完整操作
- 🌐 **局域网共享** - 同一网络下的设备均可访问
- 🛡️ **安全可靠** - 文件类型白名单、路径安全检查、中文文件名支持

## ✨ 核心功能

| 功能 | 描述 |
|------|------|
| 文件上传 | 支持点击上传和拖拽上传，实时显示上传进度 |
| 文件下载 | 一键下载，支持中文文件名 |
| 文件列表 | 实时刷新文件列表，显示文件大小和修改时间 |
| 文件删除 | 安全删除功能，带二次确认 |
| 自动重命名 | 文件重复时自动添加数字后缀 |
| 进度显示 | 上传进度条实时反馈 |
| 日志记录 | 完整的操作日志，便于排查问题 |

### 支持的文件类型

- 文档：txt, pdf, doc, docx, xls, xlsx, ppt, pptx, md
- 图片：png, jpg, jpeg, gif
- 音频：mp3
- 视频：mp4, avi, mov
- 压缩包：zip, rar, 7z
- 代码：py, js, html, css, json, xml

## 📋 系统要求

- **操作系统**：Windows / macOS / Linux
- **Python 版本**：Python 3.7 或更高版本
- **网络环境**：局域网内同一网段
- **浏览器**：Chrome / Firefox / Safari / Edge 等现代浏览器

## 🚀 安装步骤

### 方式一：自动安装（推荐）

1. 下载项目到本地
2. 双击运行 `install.bat`（Windows）
3. 等待依赖安装完成
4. 运行 `python main.py` 启动服务

### 方式二：手动安装

1. 确保已安装 Python 3.7+
   ```bash
   python --version
   ```

2. 安装依赖包
   ```bash
   pip install flask
   ```

   如果国内网络较慢，可使用清华源加速：
   ```bash
   pip install flask -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```

3. 验证安装
   ```bash
   python main.py
   ```

## 💻 使用指南

### 启动服务

```bash
python main.py
```

启动成功后会显示访问地址：

```
============================================================
🚀 局域网文件分享中心已启动！
============================================================
📍 本机访问: http://127.0.0.1:8080
🌐 局域网访问: http://192.168.x.x:8080
============================================================
```

### 访问服务

1. **本机访问**：在浏览器中打开 `http://127.0.0.1:8080`
2. **局域网访问**：在局域网内其他设备上打开显示的局域网地址（如 `http://192.168.1.100:8080`）

### 上传文件

- **方式一**：点击上传区域，选择文件
- **方式二**：直接拖拽文件到上传区域

### 下载文件

点击文件列表中的"下载"按钮即可下载

### 删除文件

点击文件列表中的"删除"按钮，确认后即可删除

### 刷新文件列表

点击文件列表右上角的"刷新"按钮更新文件列表

## 📁 项目结构

```
LocalShare/
├── main.py              # 主程序文件（Flask应用）
├── install.bat          # Windows自动安装脚本
├── README.md            # 项目说明文档
├── file_share.log       # 运行日志文件（自动生成）

```

### 目录说明

- **main.py**：核心应用代码，包含路由处理、文件管理、Web界面等
- **install.bat**：一键安装脚本，自动检测环境并安装依赖
- **files/**：所有上传的文件存储在此目录
- **file_share.log**：系统运行日志，记录所有操作和错误信息

## ⚙️ 配置说明

### 默认配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 监听端口 | 8080 | HTTP服务端口 |
| 最大文件大小 | 500MB | 单个文件上传大小限制 |
| 上传目录 | files | 文件存储路径 |
| 允许的扩展名 | 见上方列表 | 支持的文件类型 |

### 修改配置

如需修改配置，编辑 `main.py` 文件中的以下参数：

```python
UPLOAD_FOLDER = 'files'                    # 上传目录
ALLOWED_EXTENSIONS = {...}                  # 允许的文件扩展名
MAX_FILE_SIZE = 500 * 1024 * 1024          # 最大文件大小（字节）
app.config['MAX_CONTENT_LENGTH'] = MAX_FILE_SIZE
```

### 端口占用问题

如果 8080 端口被占用，可修改 `main.py` 最后一行：

```python
app.run(host='0.0.0.0', port=其他端口号, debug=False, threaded=True)
```

## 🔧 常见问题

### Q1：启动后无法访问？

**A**：检查防火墙设置，确保允许 8080 端口的入站连接。

### Q2：上传文件失败？

**A**：请确认文件类型是否在允许列表中，文件大小是否超过 500MB。

### Q3：中文文件名下载后乱码？

**A**：项目已对中文文件名进行编码处理，如仍有问题请使用 Chrome 或 Edge 浏览器。

### Q4：局域网内其他设备无法访问？

**A**：
1. 确认设备在同一局域网内
2. 检查防火墙设置
3. 确认服务启动时显示的局域网地址是否正确

### Q5：如何停止服务？

**A**：在运行 `python main.py` 的终端窗口按 `Ctrl + C` 即可停止服务。
